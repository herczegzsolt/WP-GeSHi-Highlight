== Description ==

* Syntax highlighting for **manyyy languages** (based on [GeSHi](http://qbnz.com/highlighter/)).
* **Line numbers**. Code-number displacements never happen!
* Extremely **simple usage**. But, if desired, **styles** are highly&easy configurable.
* **styles-per-block**: each code block on a single web page **can** get its own style.
* High **performance**.
* Sophisticated technology: **clean, small and valid (X)HTML output**.
* Clean and well documented source code.

== Installation ==

1. Upload the wp-geshi-highlight directory to the /wp-content/plugins directory
1. Activate the plugin through the plugins menu in WordPress

== Usage ==

See http://gehrcke.de/wp-geshi-highlight

== Frequently Asked Questions ==

= Can I change the starting line number? =

Yes, try e.g. line="13". But this [breaks](http://qbnz.com/highlighter/geshi-doc.html#starting-line-numbers) XHTML strict compliance.
